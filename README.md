# SmartShelfX – AI-Powered Inventory Intelligence 📦🤖

**SmartShelfX** is a full-stack inventory management system that provides a real-time dashboard for monitoring sales, inventory levels, and automated restocking alerts.

🔗 **Live Demo:** [https://smartshelf-lime.vercel.app](https://smartshelf-lime.vercel.app)

---

## 🚀 Key Features
* **Real-time Dashboard:** Instant visualization of total products, monthly revenue, and stock levels.
* **AI-Ready Analytics:** Predictive UI components for determining future product demand.
* **Automated Restock Alerts:** Visual indicators for items falling below safety stock levels.
* **Full-Stack Integration:** Powered by React.js with a Supabase cloud database.

## 🛠️ Tech Stack
* **Frontend:** React.js, TypeScript, Tailwind CSS
* **Backend:** Supabase (PostgreSQL)
* **Deployment:** Vercel (CI/CD)
